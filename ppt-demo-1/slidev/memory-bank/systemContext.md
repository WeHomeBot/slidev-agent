# 设计系统

## 颜色
- 主色: `--color-primary` (#3B82F6)
- 背景色: `--color-background` (#ffffff 亮色模式, #121212 暗色模式)
- 文本色: `--color-text` (#374151 亮色模式, #f3f4f6 暗色模式)
- 强调色: `--color-accent` (#10B981)
- 警告色: `--color-warning` (#F59E0B)

## 排版
- 字体: 系统默认无衬线字体
- 基础字号: 1rem (16px)
- 标题字号: 2rem (32px)
- 副标题字号: 1.5rem (24px)

## 卡片风格
- 圆角: 0.5rem (8px)
- 阴影: 0 4px 6px rgba(0, 0, 0, 0.1)
- 内边距: 1rem (16px)
- 间距: 0.75rem (12px)

## 图标
- 使用Iconify集成的图标
- 主要使用mdi和carbon图标集
- 图标大小: 1.5rem (24px)

## 动画
- 过渡效果: slide-left (默认)
- 强调动画: fade-in-up
- 动画时长: 300ms
